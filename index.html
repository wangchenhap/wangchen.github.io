如何制作个人网站

1.首先下载安装sublime，我们选择使用Boostrap建立我们的网站，可以在网上下载一个模板，右键html文件选择使用sublime打开，就可以这个模板的基础上修改代码，得到我们想要的网站样式以及内容。

2.将网页建好后，我们需要下载并安装github才能使别人能够浏览我们的网站，登上github网站注册，新建一个repo并命名，打开settings，有一个Github Pages的设置，点击Launch automatic page generator，然后点击页面最后的continue to layouts，就可以进入设置你的页面主题，选择主题，点击publish pages。

3.网站就建好了，但是现在还只是模板不是自己建造的网站，在一个盘中新建一个与repo名字相同的文件夹，右键点击 git bash here。输入git clone https://github.com/wangchenhap/your-repo-name.git克隆文件，输入git checkout -b gh-pages origin gh-pages，新建一个gh-pages分支。

4.打开建立的文件夹，删除里面的内容将自己做好的东西复制进去，输入命令git add .添加文件，输入git commit -m "your-repo-name"提交改变，然后需要告诉git添加一个远程路径，输入：git remote add origin https://github.com/yourusername/your-repo-name.get

5.最后需要上传提交的文件，输入命令git push origin gh-pages，输入账号密码后就可以在网页上搜索到你的网站了。
